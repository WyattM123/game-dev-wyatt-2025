Common Alt Codes

These are some commonly known altcodes that you can try using by holding alt and pressing any sequence of numbers on your NUMPAD.

  • Alt + 1 = ☺
  • Alt + 2 = ☻
  • Alt + 11 = ♂
  • Alt + 12 = ♀
  • Alt + 15 = ☼
  • Alt + 0169 = ©
  • Alt + 0174 = ®
  • Alt + 0153 = ™
  • Alt + 0137 = ‰
  • Alt + 0189 = ½
  • Alt + 0188 = ¼
  • Alt + 0190 = ¾
  • Alt + 0162 = ¢
  • Alt + 0163 = £
  • Alt + 0165 = ¥
  • Alt + 0128 = €
